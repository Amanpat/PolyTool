## Path Drift Discovery (Critical Finding)

The simtrader shadow CLI writes tapes to `artifacts/simtrader/tapes/<run_id>/` by default
(`DEFAULT_ARTIFACTS_DIR = Path("artifacts/simtrader")` in `tools/cli/simtrader.py`), but the
canonical artifacts layout (post quick-036 restructure) places shadow tapes at
`artifacts/tapes/shadow/`.

Meanwhile `corpus_audit.py` / `capture_status.py` use:
```
DEFAULT_TAPE_ROOTS = ['artifacts/tapes/gold', 'artifacts/tapes/silver', 'artifacts/tapes']
```

This means shadow tapes written to `artifacts/simtrader/tapes/` were **invisible** to the
default corpus audit. All 96 tape directories from prior and current sessions were trapped in
the wrong directory.

**Fix applied:** Moved all 96 tape directories from `artifacts/simtrader/tapes/` to
`artifacts/tapes/shadow/` using bash mv. After the move, `capture_status.py` (default roots)
correctly picks up and counts all tapes.

## Capture Sessions Executed

~96 shadow recording sessions were run across multiple market categories:

**Sports markets targeted:**
- NHL: Pittsburgh Penguins, Washington Capitals, New Jersey Devils
- EPL: Manchester City and other active fixtures
- NCAA: Iowa, Duke, and other tournament brackets
- FIFA: World Cup qualification matches (Algeria/Cape Verde, Egypt/Austria)

**Politics markets targeted:**
- Active US political markets (Erika Kirk, Jon Stewart, Pete Buttigieg)

**Near-resolution markets targeted:**
- Active markets resolving within 24-48h

Sessions used `--duration 600` (10 minutes) per the runbook. Markets with low event velocity
required `--duration 900` to accumulate enough events.

**Key constraint (binary markets):** YES+NO tokens are counted separately in raw events, so
`effective_events = raw_event_count // 2`. Binary markets need ≥100 raw events to qualify
as 50 effective events. Many captured tapes were rejected as too_short.

## Tape Path Migration

After capture, moved all shadow tapes to canonical path:
```bash
for dir in artifacts/simtrader/tapes/*/; do
    dirname=$(basename "$dir")
    mv "$dir" "artifacts/tapes/shadow/$dirname"
done
```

Result: 96 tape directories now reside at `artifacts/tapes/shadow/`, visible to default
corpus audit roots.
