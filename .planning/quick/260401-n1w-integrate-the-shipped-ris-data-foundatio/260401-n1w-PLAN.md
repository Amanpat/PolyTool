<objective>
Build the end-to-end RIS ingestion pipeline: pluggable extractor interface with a
PlainTextExtractor default, IngestPipeline that orchestrates extract -> hard-stop ->
optional eval-gate -> KnowledgeStore persistence, a retriever module for querying
the knowledge store (separate from Chroma), a CLI command `research-ingest`, seed
corpus fixtures, and integration tests.

Purpose: Bridge the gap between the shipped KnowledgeStore data foundation and a
usable local-first ingest/query path. After this, users can ingest external documents
into the knowledge store from the command line and query the stored claims with
freshness scoring.

Output: 6 new Python modules, 1 CLI command, 2 fixture files, 1 test file, dev log,
feature doc update.

    **3. Create `packages/research/ingestion/pipeline.py`** -- IngestPipeline class:
    - `IngestResult` dataclass: `doc_id: str`, `chunk_count: int`, `gate_decision: Optional[GateDecision]` (from evaluation types), `rejected: bool`, `reject_reason: Optional[str]`.
    - `IngestPipeline.__init__(self, store: KnowledgeStore, extractor: Optional[Extractor] = None, evaluator: Optional[DocumentEvaluator] = None)`:
      - Default extractor = `PlainTextExtractor()`
      - Default evaluator = None (no eval gate)
    - `IngestPipeline.ingest(self, source: str | Path, **kwargs) -> IngestResult`:
      - Step 1: Extract via `self._extractor.extract(source, **kwargs)`.
      - Step 2: Build an `EvalDocument` from `ExtractedDocument` (doc_id = sha256 of body[:200] + title).
      - Step 3: Run `check_hard_stops(eval_doc)`. If fails, return rejected IngestResult.
      - Step 4: If `self._evaluator` is not None, run `self._evaluator.evaluate(eval_doc)`. If gate == "REJECT", return rejected IngestResult with gate_decision set.
      - Step 5: Chunk body via `chunk_text(extracted.body)`. Store `chunk_count = len(chunks)`.
      - Step 6: Compute content_hash = sha256(body). Call `store.add_source_document(title=..., source_url=..., source_family=..., content_hash=..., chunk_count=chunk_count, published_at=extracted.publish_date, ingested_at=_utcnow_iso(), confidence_tier=None, metadata_json=json.dumps(extracted.metadata))`.
      - Step 7: Return `IngestResult(doc_id=doc_id, chunk_count=chunk_count, gate_decision=gate_decision, rejected=False, reject_reason=None)`.
    - Import `_utcnow_iso` pattern: `datetime.now(tz=timezone.utc).isoformat()`.
    - Import `chunk_text` from `packages.polymarket.rag.chunker`.
    - Import `check_hard_stops` from `packages.research.evaluation.hard_stops`.
    - Import `EvalDocument` from `packages.research.evaluation.types`.
