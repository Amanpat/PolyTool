Change the `gate` property to use weighted composite + per-dimension floor + priority tier:
```python
@property
def gate(self) -> str:
    if self.reject_reason == "scorer_failure":
        return "REJECT"
    # Load config for floors and thresholds
    from packages.research.evaluation.config import get_eval_config
    cfg = get_eval_config()
    # Floor check (skip for floor-waived tiers)
    if self.priority_tier not in cfg.floor_waive_tiers:
        for dim, floor_val in cfg.floors.items():
            if getattr(self, dim) < floor_val:
                return "REJECT"
    # Threshold check
    threshold = cfg.thresholds.get(self.priority_tier, cfg.thresholds["priority_3"])
    if self.composite_score >= threshold:
        return "ACCEPT"
    return "REVIEW"
```

Keep `total` field for backward compatibility (it becomes `simple_sum_score`). The `total` field
MUST still be populated but MUST NOT drive gate decisions. Add `simple_sum_score` as a property
alias: `simple_sum_score = total`.
