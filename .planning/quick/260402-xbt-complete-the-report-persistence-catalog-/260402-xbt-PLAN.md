    **Constants:**
    - `REPORT_LEDGER_SCHEMA_VERSION = "report_ledger_v1"`
    - `DEFAULT_REPORT_DIR = Path("artifacts/research/reports")`
    - `DEFAULT_REPORT_INDEX_PATH = Path("artifacts/research/reports/report_index.jsonl")`

    **ReportEntry dataclass:**
    ```python
    @dataclass
    class ReportEntry:
        report_id: str          # sha256(title + timestamp)[:12]
        title: str              # human-readable title
        report_type: str        # "precheck_summary" | "eval_summary" | "weekly_digest" | "custom"
        created_at: str         # ISO-8601 UTC
        artifact_path: str      # relative path to markdown file
        source_window: str      # time window covered (e.g., "7d", "30d", "all")
        summary_line: str       # one-line summary for list display
        tags: list[str]         # searchable tags
        metadata: dict          # arbitrary extra metadata
    ```

    **Functions:**
    1. `persist_report(title, body_md, report_type, source_window, summary_line, tags, metadata, report_dir) -> ReportEntry`:
       - Generate report_id from sha256(title + iso_timestamp)[:12]
       - Write markdown body to `{report_dir}/{YYYY-MM-DD}_{report_id}.md`
       - Append a ReportEntry dict as JSONL line to `{report_dir}/report_index.jsonl`
       - Create parent dirs as needed
       - Return the ReportEntry

    2. `list_reports(report_dir, window_start=None, window_end=None) -> list[dict]`:
       - Read all lines from report_index.jsonl
       - If window_start/end provided, filter by created_at (ISO string comparison)
       - Return list of dicts sorted by created_at descending

    3. `search_reports(query, report_dir) -> list[dict]`:
       - Read all entries from report_index.jsonl
       - Case-insensitive substring match on title, summary_line, and tags
       - Return matching entries sorted by created_at descending

    4. `generate_digest(window_days, report_dir, precheck_ledger_path, eval_artifacts_dir) -> ReportEntry`:
       - Compute time window (now - window_days days)
       - Load prechecks from precheck_ledger using `list_prechecks_by_window`
       - Load eval artifacts from `load_eval_artifacts` and filter by timestamp
       - Load recent reports from `list_reports` with window filter
       - Build markdown digest with sections:
         - Header with date range
         - "Prechecks" section: count by recommendation (GO/CAUTION/STOP), list ideas
         - "Evaluations" section: count by gate (ACCEPT/REVIEW/REJECT) if any
         - "Reports Generated" section: list of report titles from the period
         - "Key Observations" section: stale warning count, override count if any
       - Call `persist_report` with report_type="weekly_digest", tags=["digest", "automated"]
       - Return the persisted ReportEntry
